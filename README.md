Graph Convolutional Networks in PyTorch

## Tested Env.

- Python 3.6.9
- PyTorch 1.7.0
- CUDA 10.1

### Library Versions

```bash
numpy                 1.19.4
scipy                 1.5.4
networkx              2.5
```

## Installation

```python setup.py install```

## Usage

```bash
$ cd pygcn
$ python train_gdp.py
```
